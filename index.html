// POC

Configuration d'une infrastructure réseau en VLANs sécurisés
✓ VALIDÉ

TP Cisco Packet Tracer : schéma du réseau rapport de test de connectivité et analyse Wireshark. Mise en place d'un réseau local ou de réseau de réseaux eg. VLSM.

Segmenter le réseau de plusieurs établissements scolaires en 3 sous-réseaux (Nancy, Metz, Colmar), configurer le routage inter-cité et mettre en place des ACLs pour isoler les établissements.

Configuration des routeurs Cisco, paramétrage du routeur via RIP ou OSPF, mise en place des ACLs, vérification via ping et traceroute, analyse des trames avec Wireshark.

Mauvaise configuration de l'ordre des ACLs : l'ordre d'ajout des règles est en réalité pris en compte par le routeur

Cisco IOS Packet Tracer Wireshark OSPF RIP ACLs
// VERDICT

Cette preuve démontre ma capacité à configurer les fonctions de base d'un réseau local (AC11.03) et à identifier et résoudre des dysfonctionnements (AC11.05). J'ai compris que la segmentation réseau est un pilier de la cybersécurité : sans isolation correcte, une compromission d'un segment peut se propager à l'ensemble du SI.

Déploiement et analyse d'une solution VoIP avec Cisco CallManager
✓ VALIDÉ

Document de synthèse et d'analyse des architectures VoIP, rapport comparatif des déploiements On-Premise, et réalisation d'un schéma logique d'interaction entre les serveurs Cisco et les équipements réseau.

Évaluer la pertinence d'un déploiement de téléphonie sur site dit "On-Prem" face aux solutions Cloud pour des environnements sensibles, et modéliser l'infrastructure avec l'écosystème Cisco Unified Communications.

Analyse détaillée des avantages et des inconvénients du On-Premise. Définition de l'architecture et des rôles liant le CUCM et le CUC. Réalisation du schéma de déploiement interconnectant ces serveurs avec le réseau local.

Assurer une gestion fluide et autonome des appels sans dépendre d'une infrastructure extérieure, tout en traitant les cas de non-réponse. Résolu en modélisant précisément les flux de communication et la bascule automatique du CUCM vers le CUC pour la prise en charge de la messagerie vocale et des mises en attente en circuit fermé.

Asterisk Protocole SIP RTP/RTCP Linphone Wireshark Linux Debian
// VERDICT

Ce projet prouve ma capacité à déployer et connecter des systèmes ToIP (AC12.04) et à mesurer et analyser les signaux (AC12.01). J'ai compris le modèle en couches de la communication vocale sur IP et son impact en sécurité : les flux RTP non chiffrés sont interceptables — d'où l'intérêt du SRTP en environnement professionnel.

// POC

Application Web de gestion et de consultation des données du Vendée Globe
⚡ EN PROGRÈS

Script SQL de construction de la base de données, requêtes d'exploitation, code source Java (Servlets) du serveur REST, et code source de l'interface Web.

Concevoir une application complète permettant de stocker et d'afficher les classements des différentes éditions du Vendée Globe, ainsi que les résultats par skipper ou par voilier.

Développement complet d'une solution informatique multicouche :

  • Backend (BDD) : Modélisation et implémentation sous PostgreSQL (script vg.sql) d'une base optimisée pour gérer les skippers, les voiliers (avec historique des noms) et les courses.
  • Backend (API) : Création d'un serveur REST avec Tomcat et des Servlets Java dans NetBeans. Renvoi des données au format JSON via la bibliothèque Gson.
  • Frontend (Web) : Développement d'une Single Page Application responsive (Bootstrap) utilisant DOM Scripting et fetch pour interroger l'API (fichiers index.html, vg.js).

Rendre les pages finales dynamique afin de les adapter à l'utilisation mobile.

SQL Java API HTML CSS Full Stack
// VERDICT

Ce projet prouve ma capacité à traduire un algorithme en programme fonctionnel (AC13.03) et à m'intégrer dans un environnement collaboratif (AC13.06). L'outil développé est directement applicable dans un contexte professionnel pour l'inventaire d'un effectif ou d'équipement.

Bilan & Perspectives

J'ai découvert que j'apprends mieux par la pratique et par l'erreur. Chaque panne, chaque bug résolu a consolidé une compréhension que les cours théoriques seuls n'auraient pas pu donner. Le travail en équipe sur les SAé a aussi développé mes soft skills.
